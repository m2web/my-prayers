# Data File Schemas

All data files live in `data/` and are consumed by Hugo shortcodes. They are JSON arrays or objects. **Never commit sensitive API keys or secrets to these files.**

---

## `prayerrequests.json`

Array of prayer request entries. Used by `layouts/shortcodes/prayer-requests.html`.

```json
[
  {
    "id": 1,                          // integer, unique
    "category": "Church",             // string, used for filtering/display
    "request": "Prose prayer text...",// string, may contain \n, Scripture refs
    "userId": "m2web@yahoo.com",      // string, always the owner's email
    "dateUpdated": "2026-02-21T15:02:29.490044"  // ISO 8601 UTC datetime
  }
]
```

**Categories in use**: Church, Community Group, Coworkers, Extended Family, Federal Government, Immediate Family, Individuals, IOUS, Local Government, Me, Neighbors, Skills, State Government, World Situation

**Rules**:

- Entries without sentence-ending punctuation (`.!?`) are treated as **name lists** and skipped during AI rewording.
- Scripture references (e.g. `Colossians 3:23–24`, `Matthew 6:33`) must be preserved verbatim when rewording.
- `dateUpdated` is set automatically by `reword-prayers.py` after paraphrasing.

---

## `staff.json`

Array of church staff entries. Displayed by `todays-staff.html` shortcode, cycling one staff member per day.

```json
[
  {
    "id": 1,
    "firstName": "John",
    "lastName": "Doe",
    "title": "Senior Pastor",
    "email": "john@church.org",
    "notes": "Optional prayer context"
  }
]
```

Source of truth is `new-staff.csv`. Use `staff-csv-to-json.py` to regenerate `staff.json` from the CSV.

---

## `specialprayer.json`

Array of special (targeted) prayer entries. Generated by `email-list-to-specialprayer.py`.

```json
[
  {
    "name": "Jane Smith",
    "email": "jane@example.com",
    "notes": "Optional context"
  }
]
```

---

## `prayernotes.json`

Free-form notes for the prayer session (e.g., email-based notes). Structure mirrors the shape expected by the `prayer-requests.html` shortcode.

---

## `missionlink.json`

Single-item array with the current week's missions link.

```json
[{ "url": "https://...", "label": "Mission Name" }]
```

---

## `memoryverses.json`

Array of memory verse entries cycled day-by-day.

```json
[{ "id": 1, "reference": "Psalm 23:1", "text": "The Lord is my shepherd..." }]
```

---

## `daily-verses.json`

Large array of daily Bible verses. Cycled by the `todays-verse.html` shortcode.

---

## `esvVerses.json`

ESV Bible verse text keyed for use with the `todays-vov.html` (Voice of Voices) shortcode.

---

## `wsc.json`

Westminster Shorter Catechism Q&A array. Used by `todays-wsc.html`.

```json
[{ "id": 1, "question": "What is the chief end of man?", "answer": "..." }]
```

---

## `countries.json`

Array of countries for mission prayer, cycling daily. Used by `todays-country.html`.

```json
[{ "id": 1, "country": "Afghanistan", "url": "https://prayercast.com/..." }]
```
